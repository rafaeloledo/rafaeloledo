This site used to be a Docusaurus deployment. It now runs on a much smaller, faster stack:

- **Vite + React 19** — instant HMR, tiny bundle.
- **Tailwind v4 + DaisyUI 5** — themable, mono-first design system.
- **Markdown posts** — picked up at build time via `import.meta.glob`.
- **mise + bun** — reproducible toolchain and a fast package manager.

## Why?

The old setup was great for docs, but I wanted something terminal-flavored that
felt like *my* environment — monospace, dark by default, with a `⌘K` palette to
jump anywhere.

```ts
// Markdown is loaded statically — no server needed.
const posts = import.meta.glob('../posts/*.md', {
  query: '?raw',
  import: 'default',
  eager: true,
});
```

The output is a fully static SPA — deploy it anywhere that serves files.
